# 3D Model Renderer

Учебный проект по дисциплине **«Компьютерная графика»**: загрузка, отображение и базовое редактирование 3D‑моделей в формате OBJ с UI на JavaFX.

## Возможности
- Загрузка и сохранение моделей `.obj`.
- Отображение сцены: несколько моделей одновременно.
- Камера: вращение мышью, масштабирование колёсиком.
- Возвращение модели к исходному состоянию (reset).
- Изменение цвета модели (select color).
- Трансформации модели: rotate (вращение) / scale (масштабирование) / translate (перемещение).
- Режимы отображения: mesh (сетка), texture (текстуры), illumination (освещение).
- Источники света: добавление и удаление.
- Редактирование геометрии:
  - Удаление полигонов по индексу или диапазону.
  - Удаление вершин по индексу или диапазону (с корректным обновлением индексов).

## Управление
-	ЛКМ + движение мыши: вращение камеры вокруг сцены.
-	Колёсико: приближение и удаление.
-	Верхнее меню: File (загрузка, сохранение модели и выход из приложения), Help (информация о приложении).
-	Панель слева: добавление, удаление и выбор моделей, камер.
-	Панель справа: операции над выбранными моделями (вращение / масштабирование / перемещение, удаление полигонов или вершин и т.д.).

## Стек и требования
- Java: 21+
- JavaFX: 17.0.2
- Сборка: Maven
- Тесты: JUnit 5

## Сборка jar
```
mvn clean package
```

## Запуск
### Через Maven
```
mvn clean javafx:run
```

## Тестирование
### Запуск всех тестов:
```
mvn test
```
